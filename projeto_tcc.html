Site de cardápio de restaurante(TCC)

Trata-se de um sistema para gerenciamento de restaurante, que permite cadastrar funcionários, adicionar e editar produtos no cardápio. Além disso, os clientes podem se cadastrar, realizar pedidos e consultar o histórico de compras.

Tecnologias

O que fiz:

Neste projeto, atuei principalmente na parte funcional do site. Desenvolvi os códigos para as telas de cadastro, login, verificação, inserção de dados no banco, além da lógica de funcionamento do cardápio, do carrinho de compras e da consulta de clientes. Também fui responsável pela estilização em CSS da tela de cadastro de cargos.

Problema: Ser algo personalizável para cada restaurante

Solução: criar uma tela para cadastrar o cargo e depois colocar essas opções cadastradas no cadastro de funcionário

Problema: adicionar quantidade variada do produto

Solução:criar os botões para aumentar e diminuir a quantidade do produto, para depois poder adicionar ao carrinho
