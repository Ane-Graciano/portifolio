Painel Econômico SP – Comércio Exterior(API)

O projeto possui gráficos que evidenciam as exportações e importações dos municípios do Estado de São Paulo, possuindo diversos filtros, como valor agregado, valor FOB, kg_líquido, carga, município. O projeto é voltado para análise e comparação entre os municípios e suas cargas, para auxiliar o specialista em logística a tomar a melhor decisão em seus projetos.

O que fiz:

Neste projeto realizei a estilização da página home, com css, seguindo os parametros do figma; fiz alguns dos gráfico , como, top 5 municípios por valor agregado, o histórico de uma carga mês a mês e o gráfico que mostra a média do valor agregado mês a mês, utilizando python no google colab;

Problema: dificuldade em visualizar os dados em gráficos sobre exportação de um determinado mês e saber a melhor opção para realizar um trabalho

Solução: gerar um gráfico que mostre os munícipios que mais exportaram em um determinado mês, para ser fácil de visualizar a informação

Problema: dificuldade em visualizar os dados em gráficos sobre exportação de uma determinado carga durante o mês

Solução: gerar um gráfico que mostre uma carga específica no mês, para ser fácil de visualizar a informação
